# Flutter Todo App with Riverpod

A simple Todo app built with **Flutter** and **Riverpod** (StateNotifier + Provider).  
Supports adding, viewing, updating, and deleting todos.

## Features

- Add a new todo
- View all todos
- Update todo title
- Delete a todo
